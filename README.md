# 🍕 Pizza Delivery Analytics Dashboard (Power BI)

## 📌 Overview
This project is an end-to-end **Pizza Delivery Analytics Dashboard** built using Power BI.  
It analyzes 50,000+ records across multiple datasets including orders, users, menu items, and order items to generate actionable business insights.

The dashboard focuses on **revenue trends, delivery performance, customer behavior, and product insights** with a clean and interactive UI.

## 🧩 Data Model

The dashboard uses a relational data model with the following tables:

- `orders`
- `order_items`
- `menu_items`
- `users`
- `drivers`

Relationships were created to enable efficient cross-filtering and analysis.
